# Shortcuts
### New R + Script
Ctrl + Shift + N OR Cmd + Shift + N

### Open File
Ctrl + O OR Cmd + O

### Save
Ctrl + S OR Cmd + S

### Save All Open File
Ctrl + Shift + S OR Cmd + Shift + S

### Settings/Preferences
Cmd + ,

### Restart RStudio
Cmd + Shift + 0

### Run Snippet
Highlight  --> Ctrl + Enter OR Cmd + Enter

### Clear Console
Ctrl + L OR Cmd + L

### Render Markdown
Ctrl + Shift + K OR Cmd + Shift + K
This will create a new HTML file for the Markdown Render
